Your browser blocks API calls from local files (file:// / content://).

Easiest fix — GitHub Pages (free, 2 minutes):
1. Go to github.com and sign in (or create a free account).
2. Click New repository. Name it council. Set it to Public. Click Create.
3. Click Add file → Upload files. Upload this HTML file. Rename it to index.html before uploading.
4. Go to Settings → Pages → Source: choose Deploy from branch → main → / (root). Save.
5. GitHub gives you a URL like https://yourusername.github.io/council/. Open that on your phone.

Alternative — Netlify Drop (even faster):
Go to app.netlify.com/drop on a PC. Drag and drop this HTML file onto the page. You get an instant HTTPS URL.
